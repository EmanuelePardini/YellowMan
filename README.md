# YellowMan

## Description:
A PacMan like mini-game made to Test Unreal Engine 5 builds on Android

## Features:
- Score Component
- Invulnerability Component
- Ghost Enemies
- Character with Android Inputs
  
## How to Use:
0. Clone the repository
1. Open the uproject
